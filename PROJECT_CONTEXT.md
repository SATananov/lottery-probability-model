# Project Context — Lottery Probability Model 6/49

## Ownership and purpose

This repository is the personal experimental project of **Stefan Tananov**. It is maintained for learning, local statistical experimentation, software engineering practice, and disciplined analysis of historical Bulgarian Toto 2 — 6/49 data.

It is not a commercial forecasting product, a betting service, or a claim that fair lottery draws can be predicted. For a fair 6/49 draw, every exact six-number combination has the same theoretical probability.

## Development method

The project has been developed incrementally. Each numbered step addresses a concrete problem, adds a bounded change, and is followed by local verification. The owner reviews the behavior, accepts or rejects the result, and keeps clean checkpoints for recovery and comparison.

The project deliberately preserves:

- source modules and generated artifacts;
- historical and canonical datasets;
- verification scripts;
- operational guardrails;
- reports of successful and failed checks;
- release manifests with file hashes.

General-purpose programming and documentation tools may assist editing, but generated output is not treated as evidence by itself. Acceptance depends on reproducible local checks, dataset integrity, readable source code, and owner review.

## Interpretation limits

Statistical rankings, simulations, backtests, neural models, and portfolio builders are experimental analysis layers. They do not create a guaranteed advantage and must not be described as proof of future predictability.
