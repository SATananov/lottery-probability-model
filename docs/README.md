# Проектна документация

Тази директория събира помощната документация, която преди е била разпръсната в основната директория на проекта.

- `docs/PROJECT_CONTEXT.md` — собственост, цели и граници на интерпретацията.
- `docs/AUTHORSHIP_AND_TOOLING.md` — прозрачност за авторството и използваните инструменти.
- `docs/README_UI.md` — ръководство за локалното Streamlit приложение.
- `docs/ADVANCED_METHODS.md` — разширени статистически методи и backtesting.
- `STEP_HISTORY.md` — подробна история на завършените проектни стъпки.

Основният `README.md` остава кратка входна точка с актуалното състояние, стартиране и безопасност.

- `reports/STEP_151_2_REPOSITORY_SYNCHRONIZATION_UI_RUNTIME_REPAIRS_AND_FRESH_CLONE_INTEGRITY_CLOSURE.md` — repository synchronization, UI runtime repair and fresh-clone closure.
