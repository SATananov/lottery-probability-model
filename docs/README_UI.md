# Ръководство за Streamlit приложението

Streamlit приложението е локалният визуален панел на проекта Lottery Probability Model 6/49. То служи за преглед на данни и модели, контролирано добавяне на официални тиражи, подготовка на фиш пакети и преглед на отчети.

## Стартиране

От основната директория на проекта:

```powershell
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

Стандартният локален адрес е:

```text
http://localhost:8501
```

## Основни области

- Начало, финално табло и отчети.
- Контролирано добавяне на официален тираж.
- Подготовка на реален фиш пакет.
- Моделен фиш пакет.
- Локален дневник на изиграните фишове.
- Исторически анализи и проверка на печалби.
- Сравнение, надеждност и тегла на моделите.
- Изследователски проверки и проспективен протокол.

## Важно

Показваните класации, симулации и оценки са експериментални статистически резултати. Те не правят бъдещите тиражи предвидими и не променят теоретичната вероятност на точна комбинация 6/49.
