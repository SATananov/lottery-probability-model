# Authorship and tooling transparency

This is a personal experimental project by Stefan Tananov. Project direction, acceptance decisions, local execution and release approval remain under the author’s control.

Ordinary programming, documentation and review tools may assist editing. Generated or suggested material is not accepted as evidence by itself: repository verifiers, reproducible artifacts, checksums and local review are used before a checkpoint is accepted.

The runtime does not depend on an external generative-AI service or AI-provider SDK. The statistical and neural research modules execute locally from the source code and data included in the project.

Git authorship identifies the person who committed a change; it does not prove how every keystroke was produced. This document therefore avoids both unsupported “fully manual” claims and unsupported claims of autonomous authorship.
